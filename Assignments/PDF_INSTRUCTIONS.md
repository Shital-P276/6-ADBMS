# How to create PDF for `Assi 1`

I added a simple Python script that converts the SQL file into a PDF.

## Command

```bash
python3 Assignments/generate_pdf.py
```

## Input and output
- Input: `Assignments/Assi 1`
- Output: `Assignments/Assi 1.pdf`

## Notes
- No extra packages are required.
- If you edit `Assignments/Assi 1`, run the same command again to regenerate the PDF.
